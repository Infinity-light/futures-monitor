# 降重技巧详解

## 一、降低AIGC率

### 1. 打破排比结构

**AIGC特征句式：**
- 其一...其二...其三...
- 首先...其次...再次...
- 第一...第二...第三...

**改进方法：**

| 原句 | 改进 |
|------|------|
| 其一，从合同角度看...其二，从公共职能角度看... | 关于平台认定的性质界定，学界主要存在两种解释路径... |
| 首先，中立性保障存在欠缺；其次，参与性保障明显不足 | 中立性保障存在明显欠缺——平台与...参与性保障同样不足... |

**技巧：** 将排比改为理论对话或因果递进

### 2. 使用主观性表述

**推荐表达：**
- "笔者倾向于..."
- "研究认为..."
- "这种观点不无道理"
- "值得进一步斟酌"
- "或许可以认为"
- "可能存在的问题是"

**避免：**
- "毫无疑问"
- "显然"
- "必然"
- "绝对"

### 3. 长短句交错

**AIGC特征：** 连续长句，结构工整

**改进示例：**

原句：
> 平台认定作为投诉处理程序的结果，其形成过程在以下三个方面存在结构性不足，包括中立性保障的欠缺、参与性保障的不足以及理性化程度的局限。

改进：
> 平台认定的形成过程存在结构性不足。中立性保障有所欠缺——平台与双方均存在利益关联。参与性保障同样不足，被投诉方常在不知情中被下架。理性化程度亦有限，审核标准不统一。

### 4. 增加具体案例细节

**抽象论述：**
> 一些法院直接采信平台认定，另一些法院则否定其效力。

**增加案例细节：**
> 深圳市中级人民法院在"某文具公司诉某贸易公司商标侵权案"中直接采信了亚马逊平台的侵权认定，认为其"具有较强的专业性和时效性"；而杭州市中级人民法院在"某服装公司诉某电商公司不正当竞争案"中，则以"程序缺乏双方质证"为由否定了平台认定的效力。

### 5. 使用口语化连接语

**推荐：**
- "值得注意的是"
- "但问题在于"
- "有鉴于此"
- "换言之"
- 使用破折号、括号插入补充说明

## 二、降低查重率

### 1. 同义词替换

| 高频词 | 替换选项 |
|--------|----------|
| 困境 | 难题、症结、瓶颈、掣肘、藩篱 |
| 建构 | 构建、建立、确立、塑造、营造 |
| 效力 | 效果、效能、效应、作用、功能 |
| 认定 | 判定、确认、识别、判定、断定的 |
| 完善 | 健全、优化、改进、补足、补强 |
| 分析 | 剖析、阐释、解读、考察、探究 |
| 路径 | 进路、方案、策略、取向、选择 |

### 2. 句式重组

**主动变被动：**
- 原：法院采信平台认定
- 改：平台认定被法院所采信

**拆分长句：**
- 原：平台认定之所以能够产生推定效力，根本在于其形成过程符合正当程序的基本要求。
- 改：平台认定的形成过程符合正当程序的基本要求。这正是其能够产生推定效力的根本原因。

**调整语序：**
- 原：从权力来源的视角观察，关于平台认定的性质界定...
- 改：关于平台认定的性质界定，从权力来源的视角观察...

### 3. 增加批判性评价

**模板句式：**
- "这种担忧并非没有道理，但问题在于..."
- "表面看来似乎成立，但深入分析会发现..."
- "这一观点有其合理之处，然而..."
- "诚然...，但是..."

### 4. 引用外文文献

**作用：** 外文文献通常不在查重库范围内

**格式：**
- 直接引用外文原文（需翻译）
- 引用中文翻译版本
- 使用"参见"引入

**示例：**
> 参见[美]罗纳德·J·艾伦：《证据法：文本、问题和案例》，张保生等译，高等教育出版社2006年版，第456页。

### 5. 案例稀释法

**方法：** 增加具体案例细节，稀释理论论述

**比例建议：** 理论论述:案例分析 = 3:2 或 1:1

**操作：**
- 每个重要论点配1-2个案例
- 详细描述案例案情
- 分析裁判要旨

## 三、综合降重流程

### 初稿完成后检查清单

1. **删除所有"本文"措辞**
   - 搜索"本文认为""本文主张""本文研究发现"
   - 替换为客观表述或删除主语

2. **检查排比结构**
   - 搜索"其一""其二""其三"
   - 搜索"首先""其次""再次"
   - 改为自然过渡

3. **检查短句孤立**
   - 检查是否有单独成句的短语
   - 合并入上下文或扩展为完整论述句

4. **增加主观性表达**
   - 在关键论点处添加"笔者认为"
   - 增加批判性评价
   - 使用不确定性表达

5. **检查引注密度**
   - 每段至少1-2个脚注
   - 重要论点必须有引注支撑
   - 混合使用期刊、专著、案例、外文文献

6. **句式多样化**
   - 避免连续使用相同句式
   - 长短句交错
   - 主动被动交替

### 目标指标

| 指标 | 初稿 | 修改后 |
|------|------|--------|
| AIGC率 | 70%+ | <10% |
| 查重率 | 25%+ | <15% |
| "本文"出现次数 | 20+ | 0 |
| 脚注数量 | 30 | 80+ |
| 平均句长 | 35字 | 25字（长短交错） |
