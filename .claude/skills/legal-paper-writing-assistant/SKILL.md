# 法学论文写作助手

## 概述

本技能基于《跨境电商平台认定的证据效力规则研究——以中小企业海外商标保护为视角》的完整写作经验，为法学专业论文写作提供系统性指导。涵盖从选题聚焦到最终降重的全流程方法论。

## 核心工作流程

### 阶段一：选题聚焦

**核心原则：从宽泛到精深**

初始选题往往过于宽泛（如"跨境知识产权维权全流程"），需要收缩至单一可论证的学理点。

**收缩路径示例：**
```
跨境知识产权维权全流程
→ 平台投诉与诉讼程序衔接
→ 平台认定的程序结果效力
→ 平台认定的证据效力规则
```

**判断选题是否合适的标准：**
1. 能否用一句话概括核心论点？
2. 是否存在理论争议或实践分歧可供论证？
3. 文献支撑是否充足（核心期刊论文20+篇）？

### 阶段二：框架构建

**参照优质范文的逻辑顺延，而非直接嫁接结构**

推荐的法学论文结构：

```
一、引言（问题意识+文献综述+方法论）
二、基础理论/属性辨析（概念界定+理论框架）
三、规范分析/实践困境（现状描述+问题提炼）
四、成因剖析（制度原因+理论溯源）
五、规范建构（完善路径+具体建议）
结论
```

**避免的陷阱：**
- 二三四章全部为"困境"阐释（冗余）
- 二级标题下再分1.2.3.点（过于机械）
- 章节之间缺乏逻辑递进

### 阶段三：正文撰写

**段落式论述原则：**

| 避免 | 推荐 |
|------|------|
| 1. 直接采信与严格审查的裁判分歧 | 司法实践中的乱象，集中体现在采信标准的两极分化 |
| 其一/其二/其三 排比结构 | 使用自然过渡词语（"值得注意的是""但问题在于"） |
| 短句孤立开头："中立性保障存在欠缺" | 完整论述句："中立性保障在投诉处理程序中存在明显欠缺" |
| "本文认为""本文主张" | 删除主语，改用客观表述或"研究倾向于" |

**论证技巧：**
- 使用理论对话：呈现"合同授权论"与"公共职能论"的对立，再提出本文立场
- 增加批判性评价："这些担忧并非没有道理，但问题在于..."
- 使用不确定性表达："或许""可能""不无道理"

### 阶段四：引注规范

**引注密度要求：**
- 每个重要论点都应有引注支撑
- 每段至少1-2个脚注
- 全文脚注数量：核心期刊论文80-120个为宜

**引注格式规范：**

| 类型 | 格式示例 |
|------|----------|
| 期刊论文 | 作者：《标题》，载《期刊》年份第X期，第XX页 |
| 专著 | 作者：《书名》，出版社年份版，第XX页 |
| 法律法规 | 《法律名称》（发布日期），载《公报》 |
| 案例 | 法院（年份）案号民事判决书 |
| 二次引用 | 参见[美]罗纳德·J·艾伦等：《证据法》，张保生等译，第XX页 |
| 同前注 | 同前注[数字]，第XX页 |

**禁止事项：**
- 不使用硕士论文作为参考文献
- 不引用百度百科、知乎等非学术来源
- 外文文献需注明译者

### 阶段五：降重优化

**降低AIGC率技巧：**

1. **打破对称结构**
   - 改："从权力来源的视角观察，关于平台认定的性质界定，学界主要存在两种解释路径。'合同授权论'强调...'公共职能论'则主张..."
   - 避免："其一...其二...其三..."

2. **使用主观性表述**
   - "笔者倾向于采'有限公共职能论'"
   - "这种观点不无道理"
   - "值得进一步斟酌"

3. **长短句交错**
   - 避免连续长句（超过30字）
   - 适当插入短句形成节奏变化

4. **增加具体案例细节**
   - 将抽象论述改为"以...为例"的具体分析
   - 引用案例案号、裁判要旨

**降低查重率技巧：**

1. **同义词替换**
   - 困境/难题/症结/瓶颈
   - 建构/构建/建立/确立
   - 效力/效果/效能/效应

2. **句式重组**
   - 主动变被动："法院采信平台认定"→"平台认定被法院所采信"
   - 拆分长句：将复合句拆分为多个简单句

3. **增加批判性评价**
   - "这种担忧并非没有道理，但问题在于..."
   - "表面看来似乎成立，但深入分析会发现..."

## 输出模式

### 论文框架模板

```markdown
# [论文标题]

## 摘要
[约290字，第三视角客观概要，删除"本文"措辞]

## 关键词
[3-5个，用分号分隔]

## 一、引言
[问题背景+核心困境+文献综述+方法论]

## 二、[核心理论章节]
（一）[概念界定]
（二）[理论框架]
（三）[本文立场]

## 三、[实践/规范章节]
（一）[现状描述]
（二）[问题提炼]
（三）[典型案例]

## 四、[成因分析章节]
（一）[制度原因]
（二）[理论根源]

## 五、[完善建议章节]
（一）[总体思路]
（二）[具体路径]
（三）[配套机制]

## 结论
[核心观点总结+理论贡献+研究展望]
```

### 段落修订示例

**用户输入（问题版本）：**
> 本文认为平台认定具有复杂的性质。其一，从合同角度看，它是私主体判断；其二，从公共职能角度看，它是准行政认定；其三，本文主张分层属性说。

**输出（改进版本）：**
> 从权力来源的视角观察，关于平台认定的性质界定，学界主要存在两种解释路径。"合同授权论"强调平台认定的私法属性，认为其源于用户的意思自治，在诉讼中应作为普通证据对待；"公共职能论"则主张平台实际承担了本应由行政机关履行的公共职能，其认定具有准行政性质。两种路径的差异直接影响效力定位：依前者，平台认定仅具有私法行为的效力；依后者，则可因公共职能的行使而产生更高效力。研究倾向于采"有限公共职能论"，即承认平台在事实查明领域承担了一定的公共职能，但其法律判断权仍应受司法审查的约束。

**修订说明：**
- 删除"本文认为"，改用客观表述
- 将"其一其二其三"改为自然过渡
- 增加理论对话（两种观点对比）
- 明确本文立场及其论证依据
