# 微信公众号文章规范

## 平台特点

- **阅读场景**: 碎片化阅读，但愿意深度阅读
- **用户期待**: 有深度、有价值、值得转发
- **内容偏好**: 长文、深度分析、系统性内容

## 格式规范

### 标题

| 要素 | 规范 |
|------|------|
| 长度 | 15-30字 |
| 风格 | 正式但有吸引力 |
| 技巧 | 悬念、数字、痛点 |

**标题公式**:
- 数字+结果：「5个方法让你xxx」
- 痛点+方案：「xxx总是失败？试试这个方法」
- 身份+价值：「xxx必看：关于xxx的深度解析」

### 正文结构

```
【引言】100-200字
- 引出话题
- 建立共鸣

【正文】1200-2500字
- 使用二级标题分段
- 每段300-500字
- 重点内容加粗

【结尾】100-200字
- 总结要点
- 引导互动
```

### 排版建议

- 段落间空一行
- 重点内容**加粗**
- 适当使用引用块
- 图片配文字说明
