# 对话分析框架

## 目的

在生成文章前，深入分析对话内容，确保：
- 准确提取核心信息
- 匹配最适合的文章风格
- 保留对话中的精华观点

## 分析维度

### 1. 主题识别

| 维度 | 分析内容 |
|------|----------|
| 核心主题 | 对话讨论的中心话题 |
| 子话题 | 涉及的相关话题 |
| 关键词 | 高频出现的词汇 |

### 2. 内容类型判断

| 类型 | 特征 | 推荐风格 |
|------|------|----------|
| 技术讨论 | 代码、工具、方法论 | tutorial |
| 问题解决 | 遇到问题→分析→解决 | story, tutorial |
| 创意探索 | 头脑风暴、方案设计 | opinion |
| 知识学习 | 概念解释、原理分析 | tutorial, guide |
| 经验分享 | 个人经历、心得体会 | story, review |
| 资源推荐 | 工具、书籍、资源 | review, guide |

### 3. 信息提取

**需要提取的内容**:
- 核心观点/结论
- 关键步骤/方法
- 重要数据/案例
- 经验教训/建议
