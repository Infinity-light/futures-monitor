# 故事型文章写作指南

## 核心特征

- **目标**: 通过叙事引发情感共鸣
- **价值**: 真实感、代入感、启发性
- **拟人化表现**: 第一人称叙述、场景描写、内心独白

## 结构模板

### 微信公众号版本 (1500-2500字)

```
【标题】我是如何xxx的

【开头】(200-300字)
- 场景切入：一个具体的时刻/事件
- 制造悬念或共鸣

【正文】
## 起：问题出现 (300-400字)
- 当时的处境
- 面临的困难
- 内心的感受

## 承：探索过程 (400-500字)
- 尝试的方法
- 遇到的挫折
- 转折点

## 转：关键发现 (400-500字)
- 突破的契机
- 核心洞察
- 方法论提炼

## 合：结果与反思 (300-400字)
- 最终成果
- 经验总结
- 对读者的启发

【结尾】(100-150字)
- 升华主题
- 引发思考
```

### 小红书版本 (400-600字)

```
【标题】分享一下我xxx的经历

【正文】
事情是这样的...

（简述背景）

当时我xxx，结果xxx

后来我发现xxx

现在回头看，我觉得xxx

给大家的建议：
- xxx
- xxx

你们有过类似的经历吗？

#经验分享 #真实故事
```

## 语言风格

| 场景 | 推荐表达 | 避免表达 |
|------|----------|----------|
| 开场 | "那天..." | "本文讲述..." |
| 转折 | "直到有一天..." | "然而..." |
| 感受 | "说实话，当时挺xxx的" | "感到非常xxx" |
| 总结 | "现在想想..." | "综上所述..." |
