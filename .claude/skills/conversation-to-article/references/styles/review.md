# 种草型文章写作指南

## 核心特征

- **目标**: 分享真实体验，帮助读者做决策
- **价值**: 真实、客观、有参考价值
- **拟人化表现**: "实际用下来"、"说实话"、"优缺点都说"

## 结构模板

### 微信公众号版本 (1500-2000字)

```
【标题】用了xxx一个月，说说真实感受

【开头】(150-200字)
- 为什么关注这个
- 使用背景

【正文】
## 一、基本介绍 (200-300字)
- 是什么
- 核心功能/特点

## 二、实际体验 (500-700字)
### 优点
- 优点1 + 具体场景
- 优点2 + 具体场景

### 缺点
- 缺点1 + 影响程度
- 缺点2 + 是否可接受

## 三、适合谁 (200-300字)
- 适合的人群
- 不适合的人群

## 四、购买/使用建议 (150-200字)
- 注意事项
- 替代方案

【结尾】(100字)
- 总体评价
- 互动引导
```

### 小红书版本 (400-600字)

```
【标题】xxx真实测评｜用了一个月的感受

【正文】
先说结论：xxx

使用场景：xxx

优点：
- xxx
- xxx

缺点（也要说）：
- xxx

适合：xxx
不适合：xxx

总体评分：x/10

有问题可以问我～

#测评 #真实分享
```

## 语言风格

| 场景 | 推荐表达 | 避免表达 |
|------|----------|----------|
| 评价 | "实际用下来..." | "该产品具有..." |
| 优点 | "我觉得最好的是..." | "优势在于..." |
| 缺点 | "说实话，xxx有点..." | "不足之处是..." |
| 建议 | "如果你xxx，可以考虑" | "建议购买..." |
