# 教程型文章写作指南

## 核心特征

- **目标**: 教会读者某项技能或解决特定问题
- **价值**: 实用性强，可操作，有收藏价值
- **拟人化表现**: "我踩过的坑"、"我的经验是"、"实测有效"

## 结构模板

### 微信公众号版本 (1500-2500字)

```
【标题】如何xxx：一份实操指南

【开头】(150-200字)
- 痛点引入：描述读者可能遇到的问题
- 我的经历：简述自己如何解决这个问题
- 价值承诺：告诉读者看完能获得什么

【正文】
## 一、为什么需要xxx (200-300字)
- 背景说明
- 常见误区

## 二、准备工作 (200-300字)
- 前置条件
- 工具/资源准备

## 三、具体步骤 (800-1200字)
### 步骤1：xxx
- 操作说明
- 注意事项
- 我的经验

### 步骤2：xxx
...

## 四、常见问题 (200-300字)
- Q1: xxx
- Q2: xxx

【结尾】(100-150字)
- 总结要点
- 鼓励行动
- 引导关注/互动
```

### 小红书版本 (400-600字)

```
【标题】手把手教你xxx｜亲测有效

【正文】
先说结论：xxx

具体怎么做？往下看

1️⃣ 第一步：xxx
要点：xxx

2️⃣ 第二步：xxx
要点：xxx

3️⃣ 第三步：xxx
要点：xxx

避坑提醒：
- xxx
- xxx

我用这个方法xxx（效果描述）

有问题评论区问我～

#相关话题 #标签
```

## 语言风格

| 场景 | 推荐表达 | 避免表达 |
|------|----------|----------|
| 引入 | "很多人问我..." | "本文将介绍..." |
| 经验 | "我试过之后发现..." | "经过验证..." |
| 建议 | "建议先xxx" | "应当xxx" |
| 强调 | "这一步很关键" | "此步骤至关重要" |
