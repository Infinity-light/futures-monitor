# 攻略型文章写作指南

## 核心特征

- **目标**: 提供全面、实用的信息汇总
- **价值**: 信息密集、一站式解决、高收藏价值
- **拟人化表现**: "我整理了"、"亲测有效"、"避坑必看"

## 结构模板

### 微信公众号版本 (1500-2500字)

```
【标题】xxx完全指南：从入门到精通

【开头】(100-150字)
- 这份攻略解决什么问题
- 适合什么人看

【正文】
## 一、基础知识 (300-400字)
- 核心概念
- 必备前提

## 二、详细攻略 (800-1200字)
### 2.1 xxx
- 要点
- 注意事项

### 2.2 xxx
...

## 三、避坑指南 (200-300字)
- 常见错误1
- 常见错误2

## 四、资源汇总 (150-200字)
- 推荐工具/网站
- 延伸阅读

【结尾】(50-100字)
- 建议收藏
- 有问题留言
```

### 小红书版本 (400-600字)

```
【标题】xxx攻略｜建议收藏

【正文】
整理了一份xxx攻略，需要的收藏

一、xxx
- 要点1
- 要点2

二、xxx
- 要点1
- 要点2

避坑：
- xxx
- xxx

有用的话点个赞～

#攻略 #干货 #收藏
```

## 语言风格

| 场景 | 推荐表达 | 避免表达 |
|------|----------|----------|
| 开场 | "整理了一份..." | "本攻略包含..." |
| 列举 | "主要有这几个..." | "包括以下几点..." |
| 提醒 | "这里要注意..." | "需要注意的是..." |
| 结尾 | "有用就收藏吧" | "希望对您有帮助" |
